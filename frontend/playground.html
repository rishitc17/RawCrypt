Cipher Playground

Pick a cipher, enter a plaintext and key, and watch encryption happen. Try decrypting too — see if you can recover the original.

—

Format:

Encrypt

Decrypt

Try this

  1. Generate a random key for the cipher.
  2. Type a message in the Plaintext box.
  3. Hit Encrypt — you'll get a hex string like 5E 51 4D 4D 72 2B.
  4. Copy that hex string into the Ciphertext box.
  5. Hit Decrypt — you should get your original message back!
  6. Try changing one bit of the key and re-decrypting — see how the output is now garbage.
